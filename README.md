# Виджет проверки города · чекап VK

Проверка доступности сдачи крови по городу. Клиент вводит город, видит «да / нет».

## Файлы

- `city_checker_v2.html` — актуальный виджет в фирменном стиле Bioniq (палитра #1098D4/#05131A/#EDF8FC, Montserrat, кнопка с синим градиентом). Логика та же, что в v1.
- `city_checker_v1.html` — первая версия (нейтральный зелёный стиль), оставлена для истории.
- `cities_full.json` — полный датасет: город, регион, лаборатории (366 записей).
- `cities_full.csv` — то же в CSV (UTF-8 BOM, открывается в Excel).
- `cities_array.json` — плоский массив названий городов для виджета.

## Данные

366 уникальных населённых пунктов, 73 региона. Собраны из исходного файла с адресами точек 5 лабораторий: Инвитро, Хеликс, Медси, Хромолаб, KDL.

Формат `cities_full.json`:
```json
{ "Город": "Казань", "Регион": "Республика Татарстан", "Лаборатории": "Инвитро, КДЛ, Хеликс" }
```

## Как сделаны данные (важно для доверия к списку)

1. Город парсился из строки адреса (не из отдельного поля — его нет). Форматы в исходнике разные: «г. Казань», «город Казань», «Казань, ул...», адреса с метро, с индексом, с районом.
2. Результат сверен с эталонным справочником городов РФ (1134 города). 372 из 434 совпали.
3. Районы в виде города («Мытищинский», «Изобильненский») вручную заменены на настоящие города (Мытищи, Изобильный). Мусор («Москва и МО», «РФ», «ХМАО») удалён.
4. Осталось 23 валидных ПГТ/села, которых нет в справочнике только-городов, — оставлены как реальные точки.

## Логика виджета

- Нормализация ввода: lower, ё→е, срез небуквенных символов.
- Точное совпадение по нормализованному ключу → «да».
- Нет точного, но есть похожие (по подстроке) → «уточните город» + подсказки.
- Совсем нет → «пока не нашли, проверьте написание».
- Автоподсказка: сначала совпадения с начала строки, потом по вхождению, максимум 8.
- Клавиатура: стрелки, Enter, Escape.

## Принятые решения

- **Все 5 лабораторий.** Виджет — просто проверка «есть ли лаборатория в городе», автозапись ни при чём. Список не фильтруется по КДЛ (решение от 08.07.2026).
- **Ответ «да» показывает лаборатории города** («В городе Барнаул можно сдать анализы: Инвитро, КДЛ, Хеликс») — в v2 внутрь зашита карта город → лаборатории из `cities_full.json`.

## Нерешённые вопросы (решить перед публикацией)

1. **Часть точек не попала в список.** Где в адресе был только район без имени города, город восстановить нельзя — такие точки выпали. Клиент из такого города увидит ложное «нет».

2. **Встраивание.** Если идёт на VK-страницу, проверить, что CMS не вырезает `<script>`. Если вырезает — нужен iframe или отдельная ссылка.
